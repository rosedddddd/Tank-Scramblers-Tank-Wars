Detailed Description

Class DumbTank is an example class used to demonstrate how to use the functions available from the AITank base class.

Copy this class when creating your smart tank class.

override void DumbTank.AITankUpdate ( )

Function checks to see what is currently visible in the tank sensor and updates the Founds list.

First().Key

is used to get the first element found in that dictionary list and is set as the target, based on tank health, ammo and fuel.
